    Configuration Guide

    This page explains how to configure the application.

    Environment Variables

    All configuration is done through the .env file in the project root.

    Required Variables

    # HeyGen Configuration (REQUIRED)
HEYGEN_API_URL=https://api.heygen.com
HEYGEN_API_TOKEN=your_api_token_here
HEYGEN_AVATAR_ID=Silas_CustomerSupport_public

# Laravel Configuration (REQUIRED)
APP_KEY=base64:...
APP_URL=http://localhost:8000

    Variable Reference

    Variable Required Default Description
    HEYGEN_API_URL Yes https://api.heygen.com HeyGen API base URL
    HEYGEN_API_TOKEN Yes None Your HeyGen API key
    HEYGEN_AVATAR_ID Yes Silas_CustomerSupport_public Avatar to use
    APP_KEY Yes None Laravel encryption key
    APP_URL Yes http://localhost Application URL

    Getting Your HeyGen API Token

    Step-by-Step

    1. Go to HeyGen website 

      https://www.heygen.com/

    2. Create account or sign in

    3. Navigate to API settings 

      Dashboard → Settings → API Keys

    4. Generate new API key

    5. Click "Generate API Key"

    6. Copy the generated token

    7. Add to .env file 

      HEYGEN_API_TOKEN=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

    Keep Your Token Secret

    Never commit your API token to version control or share it publicly.

    Available Avatars

    The default avatar is Silas_CustomerSupport_public, a public interactive avatar optimized for customer support interactions.

    Changing the Avatar

    1. Find avatar ID in HeyGen dashboard
    2. Update .env: 
      HEYGEN_AVATAR_ID=your_chosen_avatar_id
    3. Clear config cache: 
      php artisan config:clear

    Laravel Configuration

    Generate Application Key

    php artisan key:generate

    This sets APP_KEY in your .env file automatically.

    Clear Configuration Cache

    After any .env changes:

    php artisan config:clear
php artisan cache:clear

    Production Configuration

    For production deployment, update these settings:

    APP_ENV=production
APP_DEBUG=false
APP_URL=https://your-domain.com

# Use HTTPS
HEYGEN_API_URL=https://api.heygen.com

    HTTPS Required

    Voice input (microphone) requires HTTPS in production browsers.

    Security Best Practices

    1. Never commit .env - Already in .gitignore
    2. Use environment variables - Not hardcoded values
    3. Rotate API tokens - Periodically regenerate
    4. Use HTTPS - Required for production
    5. Set APP_DEBUG=false - In production
