    File Reference

    This page documents all important files in the project and their purposes.

    Critical Files

    These are the most important files you need to understand:

    File Location Importance Description
    AvatarController.php app/Http/Controllers/ ⭐⭐⭐ CRITICAL All backend logic - API calls to HeyGen
    avatar-demo.blade.php resources/views/ ⭐⭐⭐ CRITICAL Complete frontend - UI, JavaScript, styling
    web.php routes/ ⭐⭐ Important Defines URL routes
    services.php config/ ⭐⭐ Important HeyGen configuration
    .env Root ⭐⭐ Important API credentials (never commit!)

    File Details

    AvatarController.php

    Location: app/Http/Controllers/AvatarController.php

    Purpose: Backend controller that handles all communication with HeyGen API.

    Methods:

    Method Purpose
    index() Returns the avatar demo view
    createSession() Creates new HeyGen session + starts streaming
    sendTask() Sends text to avatar (talk or repeat mode)
    stopSession() Stops the streaming session

    avatar-demo.blade.php

    Location: resources/views/avatar-demo.blade.php

    Purpose: Complete single-page application containing:

    • HTML structure
    • CSS styling (with dark mode support)
    • JavaScript for all interactions

    Key Sections:

    Section Purpose
    <style> CSS variables for theming, layout styles
    Video Container HTML5 video element for LiveKit stream
    Controls Sidebar Chat input, task type selector, mic button
    <script> All JavaScript logic

    web.php

    Location: routes/web.php

    Purpose: Defines all URL routes for the application.

    Route::get('/avatar', [AvatarController::class, 'index']);
Route::post('/avatar/session/create', [AvatarController::class, 'createSession']);
Route::post('/avatar/task', [AvatarController::class, 'sendTask']);
Route::post('/avatar/session/stop', [AvatarController::class, 'stopSession']);

    services.php

    Location: config/services.php

    Purpose: Configuration file that reads HeyGen settings from .env

    'heygen' => [
    'api_url' => env('HEYGEN_API_URL', 'https://api.heygen.com'),
    'api_token' => env('HEYGEN_API_TOKEN'),
    'avatar_id' => env('HEYGEN_AVATAR_ID', 'Silas_CustomerSupport_public'),
],

    .env

    Location: Project root

    Purpose: Stores sensitive configuration (API keys, database credentials)

    Security

    Never commit .env to version control! It contains your secret API token.
